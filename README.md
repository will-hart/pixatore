# tauri-game

A multiplayer HTML5 game start pack which uses:

- tauri-app to bundle
- vue for gui
- pixi.js for graphis
- colyseus for multiplayer capability

Uses yarn workspaces to manage packages.

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn dev
```

### Compiles and minifies for production

```
yarn build
```
